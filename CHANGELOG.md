# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [Unreleased]

### Added
- Core semantic diffing engine.
- OpenAPI 3.x, Swagger 2.0, Protobuf, and GraphQL support.
- Terminal, JSON, Markdown, and HTML formatters.
- GitHub Action package (`@apidiff/action`) for CI/CD integration.
- `apidiff` CLI tool with local file, URL, and Git resolution.
- Performance caching for `$ref` resolver.
- Standalone binary builds via `pkg`.
